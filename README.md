📱 iStore — Tienda de iPhones
Proyecto frontend desarrollado como pre-entrega para el curso de desarrollo web.
Sitio e-commerce dedicado a la venta de iPhones, construido con HTML5, CSS3 y Font Awesome.

🗂️ Estructura del proyecto
iStore/
├── index.html        # Página principal (Inicio)
├── contacto.html     # Página de contacto
├── styles.css        # Hoja de estilos global
└── README.md         # Este archivo

📄 Páginas
index.html — Inicio

Navbar fija con logo, links de navegación y carrito con badge contador.
Hero con título principal, subtítulo y botones de llamada a la acción.
Sección de ventajas con íconos: garantía oficial, envío gratis, cuotas y soporte.
Catálogo de productos con 6 modelos de iPhone (desde el SE hasta el 16 Pro Max), cada uno con nombre, almacenamiento, color, precio y botón de compra.

contacto.html — Contacto

Panel informativo con teléfono, WhatsApp, email, dirección y horario de atención.
Formulario con campos: nombre, apellido, email, teléfono, asunto (select) y mensaje (textarea).

🎨 Decisiones de diseño

Paleta oscura inspirada en el estilo de Apple: fondo negro #0a0a0a, acento azul #0071e3 y grises neutros.
Tipografía dual: Bebas Neue para títulos de alto impacto visual y Lato para textura de lectura cómoda.
Navbar fija con efecto glassmorphism (blur + transparencia) para mantener la navegación siempre accesible.
Cards de producto con hover animado (elevación y borde azul sutil) para mejorar la experiencia del usuario.
Diseño responsive adaptado a pantallas móviles mediante media queries y Flexbox con flex-wrap.
